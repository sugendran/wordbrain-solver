WordBrain Solver

  • No word sizes set

This was an academic exercise because I wanted to play with a trie structure. It uses a list of known words to solve the puzzle, currently this uses a limited dictionary but can be changed to use all the words in the Princeton University's WordNet dictionary. Understandably searching through the whole dictionary takes a little longer, and since the algorithm is only returning the first result it may not return a solution that the WordBrain game accepts. If you want to try it against the whole dictionary, clone the repo and run 'node lib/makedict' to replace the dictionary with the wordnet one.

PS: It's not pretty, but it does get the job done :)
